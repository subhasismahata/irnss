Theory

What is NMEA?

NMEA 0183 is a combined electrical and data specification for communication between marine electronics such as echo sounder, sonars, anemometer, gyrocompass, autopilot, GPS receivers and many other types of instruments. It has been defined by, and is controlled by, the U.S. National Marine Electronics Association. The NMEA 0183 standard uses a simple ASCII, serial communications protocol that defines how data are transmitted in a sentence from one talker to multiple listeners at a time.

Global Navigation Satellite Systems(GNSS) Receiver output message has the following six sentences: RMC, VTG, GGA, GSA, GSV and GLL.

Standard NMEA message list shown in Table1

Procedure

Read the NMEA file in any suitable software/ programming language for example in Matlab/Python. You also open a file in any text editor and try to find the GGA string. Find the NMEA string GGA . In case of GPS receiver, you get string given example
Example: $GPGGA,183417.000,04814.03970,N,01128.52205,E,0,00,99.0,495.53,M,47.6,M,*53
Each NMEA message start with ‘$’, GP: systems work in GPS mode, GGA: NMEA sentence,
Time stamp: hhmmss.sss so in this example 18 34 17.000
Latitude: DDMM.MMMMM DD: Degree 04 81 4.03970,N,
and latitude direction North /South
Longitude: longitude DDDMM.MMMMM, Degree first three digits 01128.52205,E, and direction East / West.
GPS Quality : 0 = fix not available or invalid, 1=GPS fixed or valid, 2= Differential GPS or fix valid, 6= Estimated mode
Satellite in use: example =00 in this example.
HDOP: Horizontal dilution of precision, max: 99.0 Alt: Height above mean sea level, max: 100000.0 m in this example 495.53,M Alt/val: reference unit for attitude (M=meters) in this example 47.6,M Check sum: Hexadecimal checksum in this example 53.
